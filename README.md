# IEU SFL - AI Mentor Project

This is a prototype developed for **Izmir University of Economics (IEU) - School of Foreign Languages (SFL)**.

## Project Vision
To empower English preparatory students with a personalized AI assistant that is directly integrated with the IEU SFL syllabus.

## Key Features
- **Smart Essay Analysis:** Real-time feedback on grammar and vocabulary.
- **Syllabus Integration:** Checks if the student's work meets the specific goals of the current week.
- **Progress Tracking:** Visual charts to monitor academic improvement.

## Technology Stack
- **Frontend:** HTML5, CSS3 (IEU Branding), JavaScript (Chart.js)
- **Backend Potential:** Python (FastAPI/LangChain) for RAG integration.
